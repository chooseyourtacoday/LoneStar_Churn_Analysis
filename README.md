# LoneStar Wireless — Customer Churn Analysis

This project uses Kaggle's Telco Customer Churn (IBM) dataset to simulate 
a churn analysis for a fictional telecom company called LoneStar Wireless. 
Built for portfolio purposes using Databricks, SQL, and Python.

**Dataset:** https://www.kaggle.com/datasets/denisexpsito/telco-customer-churn-ibm?resource=download

**Tools:** Databricks, SQL, Python (pandas, matplotlib, seaborn)

**Key Findings:**
- Month-to-month customers churn at 42%, compared to 3% for two-year contracts
- Single customers with no dependents represent the highest churn risk at 34%
- Recommendations include targeted discounts for long-term contracts and 
  priority outreach to high-risk single customers
